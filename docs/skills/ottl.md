# OTTL Skill

Complete reference for the OpenTelemetry Transformation Language.

For full details, see the [SKILL.md](https://github.com/henrikrexed/observability-agent-skills/blob/main/skills/otel-ottl/SKILL.md).

## Key Topics

- Editors: set, delete_key, replace_pattern, merge_maps, flatten
- Converters: string functions, type conversions, conditionals
- Contexts: resource, scope, span, spanevent, metric, datapoint, log
- Where used: transform processor, filter processor, tail sampling, routing connector
- Production patterns: PII redaction, K8s enrichment, cardinality control
