# Contributing to Observability Agent Skills

We welcome contributions to the Observability Agent Skills project!

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feat/my-feature`)
3. Commit your changes (`git commit -m 'Add my feature'`)
4. Push to the branch (`git push origin feat/my-feature`)
5. Open a Pull Request

## Guidelines

- Follow the existing code and documentation style
- Add Apache 2.0 license headers to all new files
- Ensure markdownlint passes on all markdown files
- Update documentation if your changes affect user-facing behavior

## Reporting Issues

Please use GitHub Issues to report bugs or request features.
